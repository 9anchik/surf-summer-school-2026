# Профиль и Настройки

**ID:** SCR-07  
**Тип:** Экран  
**Домен:** Mobile App / Profile  
**Приоритет:** Medium  
**Статус:** Актуален  
**Функциональные блоки:** FB-PROFILE-001

---

## Обзор

Просмотр и редактирование имени и номера телефона, выход и удаление аккаунта (FR-47..FR-49). Редактирование телефона требует повторной SMS-верификации.

---

## Навигация

Вход: Bottom Nav → Профиль.  
Исход: SCR-01 при logout/delete.

---

## Входные данные

user_profile (GET /me)

---

## Применяемые логики

LOGIC-01 (OTP) — при смене телефона; LOGIC-04 (отмена) не применяется напрямую.

---

## Используемые запросы

### GET /me, PUT /me, POST /me/delete

---

## Состояния и элементы

Просмотр данных, inline-редактирование, модалки для Logout/Delete с подтверждением.

---

## Связанные требования

### Функциональные (REQ-FUNC-*)

| ID | Название | Приоритет |
|---|---|---|
| FR-47 | Просмотр и редактирование профиля | Must |
| FR-48 | Удаление аккаунта и последствия | Must |
| FR-49 | Выход из аккаунта (logout) | Must |

### Интеграции (REQ-INT-*)

| ID | Название | API operationId (контракт) |
|---|---|---|
| REQ-INT-PROFILE | Backend Profile API | getProfile — GET /profile; updateProfile — PATCH /profile; deleteAccount — POST /profile/delete — см. /01-analysis/api/profile/profile.yaml


## Критерии приёмки

- Редактирование телефона запускает flow верификации
- Удаление аккаунта сопровождается предупреждением о последствиях
